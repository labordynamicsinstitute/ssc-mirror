outshee2


        outshee2
            See outsheet for syntax


Description


outshee2 is an enhanced version of outsheet. If filename already exists,
outshee2 appends a number and saves under that name.  outshee2 continues to
append numbers until a filename is found that does not exist or the number 20
is reached. Under Windows 3.1, only the first 6 letters of the filename are
used. Other than this, outshee2 passes all arguments and options unaltered to 
outsheet.


outshee2 is called by desrep and destest to write tab-delimited ascii files.
