help ckrename
-------------------------------------------------------------------------------


Title


    ckrename - Renames variables without losing validation
 


Syntax


        ckrename varname newvarname [, stubs(char stubs)


Description


    ckrename renames a variable, just like rename, except that it first
    checks to see if renaming the variable could disturb validation because
    of validation dependencies.
