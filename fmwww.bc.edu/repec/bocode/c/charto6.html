help for charto6
-------------------------------------------------------------------------------


Drop characteristics with names too long for saving to Stata 6 


        charto6


Description


charto6 drops characteristics whose names are too long to save to Stata 6, that
is, those with names more than 8 characters long.


Users should consider issuing save to save data sets in Stata 7 format,
including characteristics with names too long to save to Stata 6, before
issuing charto6 prior to attempting to save, old. Naturally this also applies
when other changes are made, for example, changing variable names which are too
long.


Examples


    . charto6
