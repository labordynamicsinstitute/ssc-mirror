-------------------------------------------------------------------------------
help for p2ci
-------------------------------------------------------------------------------


From P-value to Confidence Interval


    p2ci #b #pval [, dfr(#) level(#) format(%fmt) eform ]


    where


    #b is the regression coefficient
    #pval is the p-value of #b


Description


    p2ci provides the standard error and confidence limits of the
          regression coefficient when only its p-value is available.
          It is an immediate command, see help immed.


Options


    dfr(#) specifies the degrees of freedom if #b is t-distributed. The
        default is z-distributed.


    level(#) specifies the width of the confidence interval; see help level.


    format(%fmt) specifies the display format for presenting numbers.
        format(%3.2f) is the default; see help format.


    eform specifies that #b and confidence limits are to be exponentiated.


Examples


     . p2ci .90  0.107 , dfr(48)
     . p2ci .0457  0.674  
     . p2ci .0457  0.674  , eform  
     . p2ci .0457  0.674  , eform  f(%3.2f)
