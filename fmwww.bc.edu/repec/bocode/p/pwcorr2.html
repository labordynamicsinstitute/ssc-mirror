Pairwise correlations, returning results


        pwcorr2 varlist


Description


    pwcorr2 is based on official Stata's pwcorr. It returns three matrices in
    its return list: r(C), the matrix of correlations, r(Cn}, the matrix of
    sample size for each variable pair, and r(Cp), the matrix of significance
    levels, as displayed by pwcorr, sig.  The latter matrix is available even
    if the sig option is not utilized.
