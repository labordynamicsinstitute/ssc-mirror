Allows to plot the magic Triangle of Pascal




        pascal [, n(real 5)]






where :


      n is the number of desired lines 




Description


pascal is a routine to plot Pascal's Triangle. This is useful to develop (a+b)^
> n equations using Newton's binomial formulae.




Examples


        . pascal , n(10)
