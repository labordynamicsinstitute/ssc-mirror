Dissimilarity index
--------------------


    ^dissim^ varlist [^if^ exp] [^in^ range] [ ^,^ ^m^atrix^(^matname^)^ ]


Description
------------


^dissim^ displays the dissimilarity index D for each pair of variables
in varlist. If x and y are >= 0, form the proportions


    p = x / SUM x   and   q = y / SUM y


and calculate D = 1/2 SUM ( | p - q | ). D lies in [0, 1].




Options
--------


^matrix(^matname^)^ specifies that results are to be placed in matrix
    matname.




Examples
---------


        . ^dissim a b c^
