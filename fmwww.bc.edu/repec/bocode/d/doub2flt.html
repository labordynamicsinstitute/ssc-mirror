Conversion of variable type 'double' to 'float'
------------------------------------------------


    ^doub2flt^  [varlist] [^, M^indiff(real)]




Description
------------


^doub2fit^ converts variables of type ^double^ to type ^float^ if the
absolute difference between the double and float values is less than
^Mindiff^. The default value for ^mindiff^ is 0.001. Doub2flt is
useful if a large number of variables are imported into Stata from
another database in the double format and the double precision is not
needed. Under this circumstance a large amount of memory or disk space
may be wasted. ^doub2flt^ allows this space/memory to be recovered.


^doub2fit^ requires the modified ^recast^ program called ^recast2^.


Options
--------


^mindiff(^real #^)^ specifies the difference between the float and double
variables that cannot be exceeded. Only if the difference is less than this
value will the conversion will take place. 0.001 is the default value for
^mindiff^.




Examples
---------




. gen double minus = -3.33


. gen double plus = 3.33


. gen double bignum =  1234567890
