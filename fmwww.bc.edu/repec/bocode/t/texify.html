help texify
-------------------------------------------------------------------------------


Title


    texify -- Invoke texify or pdflatex command for LaTeX compiling




Syntax


        texify filename




Description


    texify provides a fast and easy way to texify or pdflatex your .tex
    documents.


    Check MiKTeX (www.miktex.org) for Windows or teTeX (www.tug.org/teTeX/)
    for MacOSX/Unix/Linux.




Examples


    cap clear*
    sysuse auto, clear
    reg mpg rep78 turn weight
    outreg2 using myfile, replace tex
    texify myfile.tex


    texify "C:\Program Files\Stata11\myfile.tex"
