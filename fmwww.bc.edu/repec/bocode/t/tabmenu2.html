-------------------------------------------------------------------------------
help for tabmenu2
-------------------------------------------------------------------------------


        tabmenu2 [varlist] [if exp] [in range] [, res(varname) typ(string)
                           row(varname) col(varname) summary(string)
                           fup(varname) perc(#) pery(#) maxval(#) fr ci rv
                           level(#) ]


 Description


tabmenu2 is the command-line version of tabmenu1, and is used
for preparing one or two-way tables of means, geometric 
means, medians, proportions, odds, or rates, from a command-line. 


Options




res        response variable
typ        type of response (metric/binary/failure/count)
row        variable determining the rows of the table
col        variable determining the columns of the table
summary      summary required according to type of response
                   binary:  prop/odds
                   metric:  mean/gmean/median
                   failure: rate
                   count:   rate
fup          variable containing follow-up time
perc       for use with proportions per #, default 100
pery       for use with rates per #,  default 1000
maxval       maximum number of values a row or column variable can have, defaul
> t 10.
fr         include frequencies in the table
ci         include confidence intervals in the table
rv         reverse rows and columns of table
level      level for confidence intervals
