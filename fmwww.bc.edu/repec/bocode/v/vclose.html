-------------------------------------------------------------------------------
help for vclose
-------------------------------------------------------------------------------


Close viewer windows


        vclose [viewername(s)]




Description


    vclose is a small convenience utility for closing viewer windows. Note
    that Stata ignores a request to close a viewer which is not open.




Examples


    close all viewer windows
    . vclose


    close viewer windows 1 through 10 (N.B. numlist allowed)
    . vclose 1/10


    close viewer window with name foo
    . vclose foo
