-------------------------------------------------------------------------------
help for xb2pi
-------------------------------------------------------------------------------


Transform the logit scores into probabilities


    xb2pi #1 #2 #3 ... [ , format(%fmt) ]


Description


    xb2pi provides the probabilities corresponding to the logit
          scores.


Options


    format(%fmt) specifies the display format for presenting numbers. The
        default is %3.2f.


Examples


     xb2pi 0.15
     xb2pi 0.15 0.10 0.20
     xb2pi 0.15 0.10 0.20 , f(%3.2f)


     sysuse cancer
     logit died
     xb2pi _b[_cons]
     mat v = e(V)
     xb2pi _b[_cons] _b[_cons]-1.96*sqrt(v[1,1]) _b[_cons]+1.96*sqrt(v[1,1])
