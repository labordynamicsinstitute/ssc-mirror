-------------------------------------------------------------------------------
help for fedit
-------------------------------------------------------------------------------


Find and edit text file from within Stata


        fedit filename [ , findfile_options ]




Description


    fedit finds a text file along the path searched by findfile, which is by
    default the adopath, and then fires up a text editor to examine that
    file. The version of fedit distributed from SSC fires up the inbuilt
    Stata do file editor doedit.




Remarks


    fedit will be most useful to those wishing to use a text editor of their
    choice from within Stata with minimal typing, in particular without
    spelling out the paths of the file and of the editor executable.


    Users are encouraged to modify fedit so that it fires up their own
    favourite text editor if they prefer. However, users are explicitly
    warned that it is emphatically not good practice to modify official Stata
    files, even if you really do know exactly what you are doing. Either way,
    users should take full responsibility for what they do with this program.




Options


    findfile_options are options of findfile.




Examples


    . fedit mydo.do
