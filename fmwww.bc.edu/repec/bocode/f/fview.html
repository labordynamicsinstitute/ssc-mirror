-------------------------------------------------------------------------------
help for fview
-------------------------------------------------------------------------------


Find and view a file


        fview filename [, asis ]




Description


    fview is a very small and simple utility which, however, can be very
    handy. fview looks for a file along the adopath and, if found, displays
    the file in the Viewer.




Options


    asis specifies that the file is to be displayed as ASCII text, regardless
        of the filename's extension. The default is to display files ending
        in .smcl and .hlp as smcl.




Saved Results


    Macros:


    r(fn)  fully qualified file name
