help fitstat_ers
-------------------------------------------------------------------------------


Title
    fitstat_ers -- computes outfit and infit statistics for the conditional
        maximum likelihood (cml) Rasch model as outlined by Linacre and
        Wright (2004). See http://www.rasch.org/rmt/rmt82a.htm.


Syntax
    fitstat_ers varlist [if] [in] , [excel(filename) latex(name)]


Description
    fitstat_ers estimates fit statistics for conditional likelihood models,
    proceeding as if Rasch model assumptions are met. The current Stata
    module raschtest calculates infit and outfit statistics transformed to be
    normal random variates; this module estimates the un-transformed infit
    and outfit statistics as outlined by Linacre and Wright (1994).
