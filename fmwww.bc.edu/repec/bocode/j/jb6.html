Jarque-Bera test for normality 
-----------------------------------------------


        ^jb6^ varname 




Description
-----------


^jb6^ calculates the Jarque-Bera asymptotic test for normality on the 
        specified variable in level form.  






Options
-------


No options available




Example
-------


. ^jb6^ varname






References
----------


  See C.M. Jarque and A.K. Bera.  1987.  "A Test for Normality of 
  Observations and Regression Residuals.  International Statistical 
  Review 55:163-172; In Damodar Gujarati.  1995.  
  Basic Econometrics.  p. 143.
