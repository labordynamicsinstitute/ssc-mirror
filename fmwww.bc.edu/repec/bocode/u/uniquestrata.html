Title


    uniquestrata -- Check that the specified variables collectively account
        for every observation




Syntax


        uniquestrata varlist




Description


    uniquestrata takes a varlist of strata variables and calculates whether
    all combinations of the levels of those strata variables add up to the
    total number of observations--no more, no less.  If not, the program
    returns error 9.
