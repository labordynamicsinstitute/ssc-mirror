help ascii
-------------------------------------------------------------------------------


Title


     Display a list of all the ASCII characters


Syntax


        ascii


Description


    ascii produces a list of all the ASCII characters from 1 to 255 in the
    results window.  Beware some ASCII characters do not have a fixed width
    and one should be the carriage return.




Example


    ascii
