help mod11
-------------------------------------------------------------------------------


Title


    mod11 -- Generador de digito verificador (DV) (Modulo 11)


Syntax


    Implementacion vectorial (gen).


        mod11 varname , generate(newvar) [replace]


    Implementacion escalar.


        mod11i #numero


    options                    Description
    -------------------------------------------------------------------------
      generate                 Nombre de la variable a crear que contendra el
                                 DV.
      replace                  En el caso de existir, reemplaza la variable
                                 especificada en el argumento generate.
    -------------------------------------------------------------------------


Description


    mod11 Aplica el algoritmo modulo 11 a un numero entero entregando como
    resultado la variable digito verificador.


    En el caso de querer calcular el DV para solo un numero, mod11i permite
    el calculo inmediato ingresando como unico argumento el numero en si.


Examples


    Generando la variable dv para la variable rut
        . mod11 rut, g(dv)


    Calculo del DV para el 12345
        . mod11i 12345
