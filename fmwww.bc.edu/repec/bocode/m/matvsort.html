-------------------------------------------------------------------------------
help for matvsort
-------------------------------------------------------------------------------


Sort vector


        matvsort input_vector output_vector [, decrease]




Description


    matvsort sorts input_vector (a 1 X c matrix or an r X 1 matrix) into
    numeric order and puts the resulting vector into output_vector.
    input_vector and output_vector may have the same name.




Options


    decrease specifies that values should be in decreasing order. The default
        is increasing order.




Remarks 


    matvsort may make it easier to identify the smallest or largest
        element(s) of a vector. If a vector b has been sorted into increasing
        order, its smallest element is accessible as b[1,1] and its largest
        as b[1,colsof(b)] if b is a row vector and b[rowsof(b),1] if b is a
        column vector.


    The corresponding names are accessible as for example


        local c1name : word 1 of `: colnames b'




Examples


    . matvsort A B
