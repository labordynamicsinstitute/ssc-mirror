help for matrsort
-------------------------------------------------------------------------------


Sort rows of matrix


        matrsort input_matrix output_matrix [, decrease]




Description


    matrsort sorts each row of input_matrix into numeric order and puts the
    resulting matrix into output_matrix.  input_matrix and output_matrix may
    have the same name.  The column names of output_matrix will be c1 up,
    regardless of input.




Options


    decrease specifies that values in each row should be in decreasing order.
        The default is increasing order.




Examples


    . matrsort A B
