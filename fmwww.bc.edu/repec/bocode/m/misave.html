-------------------------------------------------------------------------------


Save multiple datasets


        misave filename-prefix [, replace]




Description


    misave stores the multiple temporary datasets declared by the last miset
    on disk under the names filename-prefix.




Options


    replace permits misave to overwrite existing datasets.




Examples


    . miset using both
    . mido drop if sex==0
    . misave female
