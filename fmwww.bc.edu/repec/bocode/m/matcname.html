Copy matrix row and column names
--------------------------------


    ^matcname^ matrix1 matrix2


Description
-----------


^matcname^ gives matrix1 the row and column names of matrix2, provided
that the two matrices have the same dimensions. If not, an error message
will be issued.




Example
-------


    . ^matcname A B^
