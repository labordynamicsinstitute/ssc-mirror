help for modeldiag
-------------------------------------------------------------------------------


Plots for model diagnostics after regression-type commands 


    This is an index file naming and pointing to the commands in this
    package.




Plots of observed versus fitted values after model fit           ovfplot 


Plots of regress or similar fit                                  regplot 


Plot residuals versus fitted values after model fit              rvfplot2 


Plot residuals versus predictor values after model fit           rvpplot2 


Index plots of estimation results                                indexplot 


Quantile plots of fitted and residuals                           qfrplot 


Plot residual distribution after model fit                       rdplot 


Plot checking for residual heteroscedasticity after model fit    rhetplot 


Plot observed, fitted and residuals versus `time' variable       ofrtplot 


Plot residuals vs lagged residuals for time series data          rvlrplot 


Plot residual autocorrelation function                           racplot 


Plots of ANOVA fit, including interaction plots                  anovaplot
