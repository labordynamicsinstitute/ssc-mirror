-------------------------------------------------------------------------------
help for mmodes
-------------------------------------------------------------------------------
Finds the mode for a numeric varlist




        mmodes [varlist]




Description


Finds the mode for a varlist. This command was inspired by modes as this
command does not accept a varlist.


Updating this command


To obtain the latest version click the following to uninstall the old version
ssc uninstall mmodes
And click here to install the new version
ssc install mmodes


Examples


An example involving the auto.dta dataset click the commands below in order


sysuse auto,clear (<-- click this first)
mmodes mpg weight trunk foreign length
