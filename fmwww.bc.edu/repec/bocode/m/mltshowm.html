Postestimation command for mltcooksd


    Syntax


        mltshowm [,] [all]






    mltshowm is part of the mlt (multilevel tools) package.




Description


    mltshowm is an postestimation command for mltcooksd. mltshowm shows all
    models that caused Cook's D to be above the cut off value.  mltshowm uses
    Cook's D for the whole model (fixed+random part). See the help for 
    mltcooksd if you want to display selected models estimated by mltcooksd.


 
Options


    all forces mltshowm to display all estimated models.
