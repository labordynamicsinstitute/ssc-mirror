help mata mm_isconstant()
-------------------------------------------------------------------------------


Title


    mm_isconstant() -- Whether matrix is constant




Syntax


        real scalar mm_isconstant(X)




Description


    mm_isconstant() returns 1 if all elements in X are equal and returns 0
    otherwise.




Remarks


    None.




Diagnostics


    mm_isconstant() returns 1 if X is void.
