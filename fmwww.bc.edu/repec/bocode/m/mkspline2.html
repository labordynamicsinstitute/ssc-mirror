help for mkspline2
-------------------------------------------------------------------------------


Title


    mkspline2 --  Linear and restricted cubic spline construction for use
                   with the postrcspline package




Description


    mkspline2 works exactly the same as mkspline, except that it leaves
    behind information on which variables have been created, the original
    variable, and the knots as characteristics.  If other restricted cubic
    splines were created using mkspline2, than mkspline2 will overwrite
    information from the previous splines. As a consequence, the
    post-estimation commands adjustrcspline and mfxrcspline will only use
    information from the last spline created by mkspline2.


    The syntax and the options are discussed in the helpfile of mkspline.
