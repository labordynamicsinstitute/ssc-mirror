-------------------------------------------------------------------------------
help for mergevirtual                                          (March 25, 2009)
-------------------------------------------------------------------------------


Description


    Merges the final set of virtual items into the original data set, for use
        after difwithpar


    mergevirtual dataset id


    where:


        dataset is the Stata dataset to merge into


        id is the subject id variable.




Example


    run mergevirtual statadata id
