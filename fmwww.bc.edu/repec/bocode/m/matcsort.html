help for matcsort
-------------------------------------------------------------------------------


Sort columns of matrix


        matcsort input_matrix output_matrix [, decrease]




Description


    matcsort sorts each column of input_matrix into numeric order and puts
    the resulting matrix into output_matrix.  input_matrix and output_matrix
    may have the same name.  The row names of output_matrix will be r1 up,
    regardless of input.




Options


    decrease specifies that values in each column should be in decreasing
        order. The default is increasing order.




Examples


    . matcsort A B
