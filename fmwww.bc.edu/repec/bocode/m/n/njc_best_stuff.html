Description 


    This is a list of what I regard as my best Stata packages or commands
    that are in the public domain.  If a help file is not accessible to you,
    use findit to find out where the files are. The help files carry details
    on people who kindly reported bugs or made useful suggestions. A fuller
    list is available at NJC stuff.
