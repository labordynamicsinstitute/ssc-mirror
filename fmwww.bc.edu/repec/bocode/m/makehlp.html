Title
    makehlp -- Automatically create a help file


Syntax
        makehlp [, options]


    options               Description
    -------------------------------------------------------------------------
    Main
      file(string)         specifies the adofile excluding the .ado
                            extension.
      replace              specifies that the old help file is replaced.
    -------------------------------------------------------------------------


Description
    makehlp creates help files automatically from a Stata ado file. This
    should save developers some time as the helpfiles are not exactly
    WYSIWYG.


Options
        +------+
    ----+ Main +-------------------------------------------------------------
    file(string) specifies the adofile excluding the .ado extension.


    replace specifies that the old help file is replaced.




Examples


    makehlp, file(makehlp) replace
