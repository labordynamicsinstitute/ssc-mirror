help soepdo
-------------------------------------------------------------------------------


Title


    soepdo -- Retrieval of SOEP drop-outs made easy






Syntax


    soepdo [namelist] , soepdir() yperg() [erhebj()]






Description


    soepdo is intended for people who need to build datasets which contain
    drop-outs from the German Socio-Economic Panel Study (SOEP Group, 2001).
    In the current version only the retrieval of variables in YPBRUTTO is
    supported.


    Where appropriate the options are named as the variables in YPBRUTTO
    which are used as criterium for inclusion of observation in your dataset.






Options


    namelist is optional. One may specify variables from YPBRUTTO here which
        are than included in the new dataset. If no variables are given, all
        variables from YPBRUTTO are included. PERSNR, ERHEBJ and YPERG are
        always included.


    soepdir is mandatory. It tells soepdo the path to the directory where
        SOEP files are stored.


    yperg is manadatory. Here one specifies the reasons for drop-outs in
        terms of YPERG (see Frick and Haisken-DeNew, 2005). If no integer is
        given, no observations will be in your dataset.


    erhebj is optional. Some drop-outs can occur multiple times per person,
        e.g. moving abroad. By default, soepdo will keep all observations but
        one may specify e(last) or e(first) to keep only the first or the
        last drop-out in terms of ERHEBJ.
