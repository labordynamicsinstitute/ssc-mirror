help spmat use
-------------------------------------------------------------------------------


Title


    spmat use -- Load an spmat object into memory




Syntax


        spmat use objname using filename [, replace]




Description


    spmat use creates the spmat object objname from the file filename
    previously saved with spmat save.  replace permits spmat use to overwrite
    objname if it already exists.




Example


    Setup
        . clear all
        . set memory 50m
        . use pollute
        . spmat contiguity cobj using pollutexy, id(id) normalize(minmax)
        . spmat save cobj using cobj.spmat
        . spmat drop cobj


    Create the spmat object cobj from the file cobj.spmat
        . spmat use cobj using cobj.spmat




Also see
