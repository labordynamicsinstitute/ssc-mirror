help for savesome
-------------------------------------------------------------------------------


Save subset of data


        savesome [varlist] [if exp] [in range] using filename [,
                 save_options]




Description 


    savesome saves part of the data in memory to filename.  If filename is
    specified without an extension, .dta is used.




Options 


    save_options are options of save. See help on save.




Examples


    . savesome if foreign using foreign.dta


    . savesome mpg weight make using small


    . savesome mpg weight make using small6, old
