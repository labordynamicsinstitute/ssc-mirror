help spmat drop
-------------------------------------------------------------------------------


Title


    spmat drop -- Drop an spmat object from memory




Syntax


        spmat drop objname




Description


    spmat drop drops the spmat object objname from memory.




Example


    Setup
        . spmat use cobj using pollute.spmat


    Drop the spmat object cobj from memory
        . spmat drop cobj




Also see


    Online:  spmat, spreg, spivreg, spmap, shp2dta, mif2dta (if installed)
