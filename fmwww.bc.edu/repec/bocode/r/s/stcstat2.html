Title


    stcstat2 -- Harrell's concordance index for flexible parametric models


Syntax


        stcstat2




Description


    stcstat2 calculates and reports Harrell's c-index after fitting a model
    with stpm2.


    See estat concordance for further information. Note, however, that
    stcstat2 supports only the all and noshow options available with estat
    concordance.




Saved results


    estat concordance saves the following in r():


    Scalars   
      r(N)           number of observations
      r(n_P)         number of comparison pairs
      r(n_E)         number of orderings as expected
      r(n_T)         number of tied predictions
      r(C)           Harrell's C coefficient
      r(D)           Somers' D coefficient for Harrell's C
