Description


    The functions above are used by somersd to calculate jackknife
    pseudovalues for sample means, Hoeffding U statistics and Von Mises V
    statistics, and by censlope to carry out bracket convergence in the
    iterative estimation of percentile slopes.  The pseudovalues are based on
    concordance-discordance difference totals, which can be calculated
    efficiently using balanced binary search trees.  The program somersd uses
    these pseudovalues to calculate delta-jackknife confidence intervals for
    Somers' D or for Kendall's tau-a.
