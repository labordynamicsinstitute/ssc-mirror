help hlpedit
-------------------------------------------------------------------------------


Title


     hlpedit -- Edit help file in do-file editor


Syntax


        hlpedit cmdname [{.hlp|.sthlp}]




Description


    hlpedit opens the help file for cmdname in Stata's do-file editor. The
    program might be useful for users who wish to document their ado files.




Example


        . hlpedit generate
        
        . hlpedit su
        
        . hlpedit tab.hlp
