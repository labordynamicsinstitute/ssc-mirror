        +--------+
    ----+ Syntax +-----------------------------------------------------------


          lmcovnlsur


        +-------------+
    ----+ Description +------------------------------------------------------


  lmcovnlsur computes Breusch-Pagan LM Diagonal Covariance Matrix Test after:
   - (NL-SUR) Seemingly Unrelated Regression nlsur for sets of equations.
 


        +---------------+
    ----+ Saved Results +----------------------------------------------------


  lmcovnlsur saves the following in r():


   r(lmcov)        Lagrange Multiplier LM Test
   r(lmcovp)       Lagrange Multiplier LM Test P-Value
   r(lmcovdf)      Chi2 Degrees of Freedom

        +----------+
    ----+ Examples +---------------------------------------------------------


 clear all


 sysuse lmcovnlsur.dta , clear


* (1) NL-SUR Model:


 nlsur (y1={B10}+{B11}*y2+{B12}*x1+{B13}*x2) (y2={B20}+{B21}*y1+{B22}*x3+{B23}*
> x4)


 lmcovnlsur


 return list


* (2) SUR Model:


 sureg (y1 y2 x1 x2) (y2 y1 x3 x4)


 lmcovnlsur


 return list
