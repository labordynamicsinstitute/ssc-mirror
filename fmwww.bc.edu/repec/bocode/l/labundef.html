help labundef
-------------------------------------------------------------------------------


Title


     labundef -- List undefined value lables




Syntax


        labundef [, detach[(varlist)] ]




Description


    labundef lists undefined value labels attached to variables in the
    current dataset.




Options


    detach[(varlist)] detaches undefined value labels from varlist. If no
        varlist is specified any undefined value label is detached from any
        variable using it.




Example


        . sysuse auto
        . label values rep78 foo
        . labudef




Saved results


    labundef saves the following in r():


    Macors
      r(labels)      undefined value lables
      r(lblname)     varlist that has lblname attached
