Left-justifying string variables for output
-------------------------------------------


        ^ljs^ varlist


Description
-----------


^ljs^ takes each string variable within varlist and left-justifies it by
trimming leading and trailing spaces and padding with as many spaces on
the right as are needed to make each value exactly the same length.


The result may be useful for listing or outfiling. However, it is not
recommended that variables are saved in this form.


Example
-------


    . ^ljs make^
