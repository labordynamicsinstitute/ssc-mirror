    lmcovxt: Breusch-Pagan LM Diagonal Covariance Matrix Panel Test


        +-------------------+
    ----+ Table of Contents +------------------------------------------------


     Syntax
     Options
     Description
     Saved Results

 *** Examples


     Author


        +--------+
    ----+ Syntax +-----------------------------------------------------------


    lmcovxt depvar indepvars [if] [in] [weight] , id(#) [noconstant vce(
    vcetype) level(#)]


        +---------+
    ----+ Options +----------------------------------------------------------




      * id(#) Number of Cross Sections in the Model
 


      noconstant     suppress constant term


      vce(vcetype)   ols, robust, cluster, bootstrap, jackknife, hc2, hc3


      level(#)       set confidence level; default is level(95)


        +-------------+
    ----+ Description +------------------------------------------------------


  lmcovxt computes Breusch-Pagan LM Diagonal Covariance Matrix Test for panel
  data.
 
  Ho: Null hypothesis of diagonal covariance matrix means no cross-section
  correlation. LM test has an asymptotic Chi2 distribution with [N(N-1)/2]
  degrees of freedom.


        +---------------+
    ----+ Saved Results +----------------------------------------------------


  lmcovxt saves the following in r():


   r(lmcov)        Lagrange Multiplier LM Test
   r(lmcovp)       Lagrange Multiplier LM Test P-Value
   r(lmcovdf)      Chi2 Degrees of Freedom
