Title


    cem -- Keep only complete cases


Syntax


        imbalance varlist










Description


    listwise removes observations that are have missing values in the
    varlist.


Arguments


        +------+
    ----+ Main +-------------------------------------------------------------


    varlist is a list of variables to be use for analysis.
