Delete value labels 
-------------------


    ^labdel^ valuelabelname ^, d^elete^(^numlist^)^ [ ^l^ist ] 


Description
-----------


^labdel^ deletes specified value labels from value label valuelabelname.


The ^label define^ ... ^, modify^ command that deletes the labels is 
echoed to the monitor. 




Options
-------


^delete(^numlist^)^ specifies a numlist of integers whose 
    value labels are to be deleted. This is a required option. 


^list^ invokes ^label list^ valuelabelname after the deletions. 
    
    
Examples
--------


    . ^labdel foo, d(0 1)^
