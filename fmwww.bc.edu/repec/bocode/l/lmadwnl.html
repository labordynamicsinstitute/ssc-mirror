        +-------+
    ----+ Title +------------------------------------------------------------


lmadwnl: NLS Autocorrelation Durbin-Watson Test at Higher Order AR(p)


        +-------------------+
    ----+ Table of Contents +------------------------------------------------


     Syntax
     Options
     Description
     Saved Results
     References


 *** Examples


     Authors


        +--------+
    ----+ Syntax +-----------------------------------------------------------


   lmadwnl depvar [if] [in] [weight] , fun(expression) lags(#)
 
     [ initial(init_val) variables(varlist) vce(vcetype ]


        +---------+
    ----+ Options +----------------------------------------------------------


  lags(#)              Lag Length Order


  depvar               Dependent Variable


  variables(varlist)   Independent Variables in model


  initial(init_val)    Parameters (initial) Starting Values


  fun(expression)      RHS Mathematical Expression


    SE/Robust
    vce(vcetype)      vcetype may be gnr, robust, cluster clustvar,
                        bootstrap, jackknife, hac kernel, hc2, or hc3


        +-------------+
    ----+ Description +------------------------------------------------------


    lmadwnl computes NLS Autocorrelation Durbin-Watson Test at Higher Order
    AR(p).


        Ho: No Autocorrelation - Ha: Autocorrelation
        - Rho Value for
        - Durbin-Watson Test


        +---------------+
    ----+ Saved Results +----------------------------------------------------


lmadwnl saves the following in e():


   e(rho#)         Rho Value for AR(i)
   e(lmadw#)       Durbin-Watson Test AR(i)

    ----+ Examples +---------------------------------------------------------


clear all


sysuse lmadwnl.dta , clear


gen ly=ln(y)


lmadwnl ly , fun({B0}+{B1}*k+{B2}*l)


lmadwnl ly, fun({B}-({H}/{R})*ln({D}*l^(-{R})+(1-{D})*k^(-{R}))) in(B 1 H 1 R 1
>  D 0.5) lag(4)
