help labvars
-------------------------------------------------------------------------------


Title


     labvars -- Label a list of variables


Syntax


        labvars varlist \ ["label" ["label"] ...] [, options]


    options                Description
    -------------------------------------------------------------------------
      unique               use last label once only
      test                 test that number of labels match number of
                             variables
      echo                 echo the label variable commands
    -------------------------------------------------------------------------


Description


    labvars attaches a list of labels to varlist. If no labels are specified,
    existing variable labels are removed. If fewer labels than variables are
    specified, the last label will be used as variable label for each
    additional variable.


Options


        +---------+
    ----+ Options +----------------------------------------------------------


    unique prevents labvars from using the last label for each additional
        variable, if fewer labels are specified than varibales are in 
        varlist.


    test tests that the number of labels equal the number of variables in 
        varlist. If too few or too many labels are specified, an error
        message will be returned.


    echo echos the label variable commands that are used.


Examples


        . sysuse auto ,clear
        (1978 Automobile Data)


        . describe make rep78 foreign


                      storage  display     value
        variable name   type   format      label      variable label
        ----------------------------------------------------------------------
        make            str18  %-18s                  Make and Model
        rep78           int    %8.0g                  Repair Record 1978
        foreign         byte   %8.0g       origin     Car type


        . labvars make rep78 foreign \ Make "Repair Record in 1978" Type


        . describe make rep78 foreign


                      storage  display     value
        variable name   type   format      label      variable label
        ---------------------------------------------------------------------
        make            str18  %-18s                  Make
        rep78           int    %8.0g                  Repair Record in 1978

        . labvars _all \ "one label fits all"
