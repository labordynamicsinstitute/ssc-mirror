help ldir


-------------------------------------------------------------------------------


Title


    ldir Clickable list of folder names


Syntax
        ldir [ stub ] [ , hidden ]


Description


    ldir lists the names of folders or directories in compact form and allow
    to click on the names to make it the working directory.


    If stub is specified, only folders with names containing stub are listed.


    If Nick Winter's user written program fastcd has been used to define
    bookmarks, those bookmarks are also shown. In addition ldir shows the
    clickable link [+] for adding the current working directory to the list
    of bookmarks. If the current working directory is already bookmarked the
    clickable link [-] can be used to remove it from the list of bookmarks.


    The list of foldernames is returned in r(folders), so long as that is not
    empty.


Options


    hidden By default, ldir does not list folder names starting with ".";
        these are the so called "hidden" folders on Linux/UNIX systems. The
        option hidden lists these folders as well.


Example(s)


    . ldir
