        +-------+
    ----+ Title +------------------------------------------------------------


 lmnjbnl: NLS Non Normality Jarque-Bera Test


        +-------------------+
    ----+ Table of Contents +------------------------------------------------


     Syntax
     Options
     Description
     Saved Results
     References


 *** Examples


     Authors


        +--------+
    ----+ Syntax +-----------------------------------------------------------


   lmnjbnl depvar [if] [in] [weight] , fun(expression)
 
     [ initial(init_val) variables(varlist) vce(vcetype ]


        +---------+
    ----+ Options +----------------------------------------------------------


  depvar               Dependent Variable


  variables(varlist)   Independent Variables in model


  initial(init_val)    Parameters (initial) Starting Values


  fun(expression)      RHS Mathematical Expression


    SE/Robust
    vce(vcetype)      vcetype may be gnr, robust, cluster clustvar,
                        bootstrap, jackknife, hac kernel, hc2, or hc3


        +-------------+
    ----+ Description +------------------------------------------------------


    lmnjbnl computes Non Linear Least Squares Non Normality Jarque-Bera Test.


        Ho: Normality - Ha: Non Normality
        - Jarque-Bera LM Test


        +---------------+
    ----+ Saved Results +----------------------------------------------------


    lmnjbnl saves the following in e():


*** Non Normality Tests:
   e(lmnjb)        Jarque-Bera LM Test
   e(lmnjbp)       Jarque-Bera LM Test P-Value
