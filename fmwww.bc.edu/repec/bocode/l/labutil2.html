help labutil2
-------------------------------------------------------------------------------


Title


     labutil2 -- Manage value and variable labels




Description


    labutil2 is a bundle of programs to manage value labels, variable labels
    and more. Programs are listed below.


    Some of the programs modify existing value labels and/or variables that
    are not explicitly specified. This behavior violates Stata conventions.
    Read the help files carefully, before using the programs.




        +-------+
    ----+ Index +------------------------------------------------------------


    New value labels


      labascii    Define value labels from ASCII (text) file
      labcpy      Modify copy of value label
      labdeval    Define value labels matching variable names
      labvalpool  Pool (sets of) value labels




    Modify existing value labels


      labmm       Modify multiple value labels
      labren      Rename value labels
      labvalch3   Change text in value labels




    Value label information


      dslab       Search for string in value labels
      labellist   List value labels and more
      uselab      List value labels and variables using them




    Variable labels


      labvars     Label list of variables




    Create variables


      labgen2     Create labeled variable




    Change variables


      labmv       Change numeric values to extended missing values preserving
                    value labels
      labmvs      Change numeric values associated with value labels to
                    extended missing values
      labrecode   Modify value labels and recode variables they are attached
                    to




    Data cleaning


      labdu       Drop unused value labels from memory or delete values from
                    value labels
      labundef    List undefined value labels in current dataset
      labvalcl    Clear value labels of null strings




    Programming


      labunab     Unbbreviate value label list
