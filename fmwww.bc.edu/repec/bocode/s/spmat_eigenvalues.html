help spmat eigenvalues
-------------------------------------------------------------------------------


Title


    spmat eigenvalues -- Add eigenvalues of the spatial-weighting matrix W to
                           an spmat object




Syntax


        spmat eigenvalues objname [, options]




    options                         Description
    -------------------------------------------------------------------------
      eigenvalues(vecname)          Mata vector containing eigenvalues
      replace                       replace existing eigenvalues
    -------------------------------------------------------------------------


    
Description


    spmat eigenvalues calculates the eigenvalues of the spatial-weighting
    matrix W contained in the spmat object objname and stores them in
    objname.  Having precomputed eigenvalues in objname will speed up the
    estimation command spreg ml.




Options


    eigenvalues(vecname) specifies a Mata vector containing the eigenvalues
        of W.  The vector must be stored as a complex rowvector.


    replace allows the existing eigenvalues to be overwritten.




Example


    Setup
        . spmat use cobj using pollute.spmat


    Add eigenvalues to the spmat object cobj
        . spmat eigenvalues cobj
