Spearman-Brown reliability


        sbri [, obs(#) format(%fmt) ]


Description


    sbri calculates the Spearman-Brown reliability. It must be used after
    xtreg with the mle (maximum-likelihood random-effects estimator) option.
    It is an immediate command.


Options


    obs(#) specifies the number of repeated measurements. The default is
        e(g_avg).


    format(%fmt) specifies the display format for presenting numbers. The
        default is %3.2f.


Examples


    use http://www.stata-press.com/data/r9/auto7
    loneway mpg manufacturer_grp if nummake == 4
    xtreg mpg if nummake == 4 , i(manufacturer_grp) mle
    sbri
