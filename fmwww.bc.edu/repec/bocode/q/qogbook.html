help qogbook
-------------------------------------------------------------------------------


Title


    qogbook -- Opens the codebook


Syntax
        qogbook


Description
    qogbook opens the correct codebook for the QoG data currently stored in
    memory. The codebook is opened as a .pdf in the default web browser.
