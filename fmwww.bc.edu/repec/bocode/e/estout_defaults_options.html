style(style) specifies a "style" for the output table. defaults(style) is
        a synonym for style(style). A "style" is a named combination of
        options that is saved in an auxiliary file called estout_style.def.
        estout is already equipped with four such files. The four styles and
        their particulars are:


            settings                 styles
                         tab     fixed   tex     html
            -----------------------------------------------
            begin                                <tr><td>
            delimiter    _tab    " "     &       </td><td>
            end                          \\      </td></tr>
            varwidth     0       12      12      12
            modelwidth   0       12      12      12
            abbrev       off     on      off     off


        The tab style is the default. Use estoutdef to view or edit defaults
        files. To make available an own set of default options, edit one of
        the existing files and save it as estout_newstyle.def somewhere in
