-------------------------------------------------------------------------------
help for estoutdef
-------------------------------------------------------------------------------


Utility to view and edit estout's defaults files




        estoutdef style [ , { view | edit | starbang } ]




    where style is tab | tex | html | mystyle




Description


    estoutdef is a small utility to view and edit estout's defaults. The
    defaults file associated with style will optionally be typed to the
    results window, viewed in the viewer, or opened in the do-file editor.




Options


    view specifies that the defaults file be displayed in the Viewer. The
        default is to type the file to the results window.


    edit specifies that the defaults file be opened in the Do-file Editor.


    starbang specifies that only the lines beginning with the characters "*!"
        be typed. Such comment lines are used to indicate the version number
        of the defaults file.




Saved results


    r(fn)        fully qualified name of the defaults file
