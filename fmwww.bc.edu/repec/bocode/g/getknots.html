help for getknots
-------------------------------------------------------------------------------


Title


    getknots -- Stores the location of knots in a local macro.


Syntax


        getknots


Description


    getknots can be used after the mkspline command using the cubic option.
    It stores the location of the knots in a local macro called `knots'.




Example




    sysuse uslifeexp, clear
    drop if year == 1918 /// spanish flu
    mkspline ys = year, cubic
    getknots
    di "`knots'"
