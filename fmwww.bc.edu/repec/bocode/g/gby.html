subset graphs and calculates in the same output
------------------------------------------------


  ^gby^ varname^:^ command [, ^hi^lo ^gn^ame(newname) ^nol^ab 
   original-options]


Description
------------


^gby^ makes subgroup graphs including -kdensity- in the same plot. 
It also works with -means-, -centile- and -summarize- for subgroup 
calculations. 


options 
---------
^hi^lo: for high-low bar
^gn^ame: specify label names    
^nol^ab: suppress automatic label


Examples
---------
    ^gby sex: graph height age, ylab  xlab by(race) s(po)^
    Scatter plots: both sex in the same plot separately for race


    ^gby sex: graph height, bar mean ylab by(race)^
    One bar plot height (mean) vs. race clustered by sex


    ^gby sex: graph height, box mean ylab by(race)^
    One box plot height vs. race clustered by sex


    ^gby sex: graph y hi lo x,  hilo  xlab(1(1)4) ylab^ 
    One high-low charts: both sex in the save plot.
    
    ^gby sex: kdensity height, n(80)^


    ^gby agegr: means height weight sbp^
