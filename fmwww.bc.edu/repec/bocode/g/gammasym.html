help for gammasym
-------------------------------------------------------------------------------


The symmetric gamma function


        gamma list




    varlist is a list of the values to use.


Description


    gammasym compute the values of the symmetric gamma function for all the
    possible level.


Example


        . gammasym -2.1 -1.5 -.7 -.6 1.1 1.00024 2.232447


        . gammasym .3 .4 -.1 -.5 0 .6




Outputs


    r(gamma#) return the value of the symmetric gamma function at the level
    #.
