isco6888


        isco6888 newvarname , isco(varname)


Description


The isco package consists of a series of programs to recode ISCO-68
(International Classification of Occupations) codes into other occupational
scales. Use the isko package to recode ISCO-88 occupational codes.  These
programs are Stata versions of the SPSS recode files by Harry B.G. Ganzeboom
and Donald J. Treiman, which are available at http://www.fss.uu.nl/soc/hg/ismf.


isco6888 transforms ISCO-68 codes into ISCO-88 codes; cf ILO (1990).  The
argument for the isco commands must be a new variable name in which the recoded
occupational scale is to be placed. The isco option is used specify the
variable containing the ISCO-68 occupational codes.


Options


isco specifies the variable to be recoded.  This must be a 4 digit integer
    containing ISCO-68 occupational codes.


References


Ganzeboom, H.B.G., de Graaf, P.M. & D.J. Treiman. 1992.  A Standard
    International Socio-Economic Index of Occupational Status.  Social Science
