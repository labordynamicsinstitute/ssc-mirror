help for betacoef
-------------------------------------------------------------------------------


Retrieve beta coefficients after regress, ivreg, ivreg2


        betacoef


Description 


    betacoef computes the beta coefficients from the previous regression
    (estimated by regress, ivreg or ivreg2) and returns them in r(beta). The
    regression may use weights (except probability weights (pw), which are
    not supported by summarize). As only the original regression coefficient
    point estimates are used in the computation, the form of standard errors
    does not matter. For compatibility with version 7, the beta coefficient
    for the constant term (if present) is set to zero rather than missing.
