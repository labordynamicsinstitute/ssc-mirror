help cdout


-------------------------------------------------------------------------------


Title


    cdout -- Opens the current directory for your viewing pleasure




Syntax


    cdout




Description


    cdout provides a fast and easy way to open the current directory (cd)
    for your viewing pleasure.


    It will also place clickable interface on the screen, which may work
    for Mac and Unix versions.




Examples


    cdout




Remarks


    May work with much earlier version than Stata 7.
