Title


    keepvar --- Keep a list of variables in the order of their appearance in
        the list


Syntax


    keepvar varlist, [asis]


  -----------------------------------------------------------------------------


Description


    keepvar, a wrapper for Stata's keep and order, keeps variables in the
    order of their appearance in a varlist.


Options for keepvar


        +---------+
    ----+ Options +----------------------------------------------------------


    asis specifies that the variables should be kept in the order they appear
    in the varlist after keepvar. If this option is not specified keepvar
    behaves as keep. If all variables in the dataset are specified after
    keepvar and the asis option is specified, then keepvar behaves as order
