-------------------------------------------------------------------------------
help for kountryadd                                  (author: Rafal Raciborski)
-------------------------------------------------------------------------------


Description


The kountryadd command adds alternative country name spellings to the kountry
   command.




Syntax


   kountryadd "failed name" to "standardized name" add


     where


   failed name represents country name that failed to standardize after the
     kountry country_var, from(other) command was issued


   standardized name represents one of the standardized country names


   Double quotes are required.




Examples


  . kountryadd "Netherlands, The" to "Netherlands" add


  . kountryadd "korea (south)" to "South Korea" add




Also see
