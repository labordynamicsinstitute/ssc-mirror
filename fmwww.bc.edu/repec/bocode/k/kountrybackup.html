-------------------------------------------------------------------------------
help for kountrybackup and kountryrestore            (author: Rafal Raciborski)
-------------------------------------------------------------------------------


Description


Use kountrybackup to back up the existing custom dictionary of country names
   created by kountryadd.  Use this command before reinstalling or updating the
   kountry command, as this process will overwrite your custom dictionary.  Use
   kountryrestore after reinstalling or updating kountry to restore the saved
   dictionary.




Syntax


   kountrybackup [, replace]


      replace permits kountrybackup to overwrite an existing backup file




   kountryrestore


      restores previously saved dictionary


Examples


  . kountrybackup, replace
    (reinstall kountry)
  . kountryrestore




Also see
